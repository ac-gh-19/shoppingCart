# Shopping Cart

### Live Demo
[shopping-cart.vercel.app](https://shopping-cart-roan-ten.vercel.app/)

## Project Description
Mock shopping site built to practice React fundamentals and routing using React Router. Users can browse store items, adjust quantities, and manage their cart before proceeding to checkout. The app maintains state across pages, dynamically updates totals, and utilizes core concepts such as shared state, controlled inputs, and navigation.
